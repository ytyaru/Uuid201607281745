# このソフトウェアについて #

Uuid201607281745は私個人が学習目的で作成したソフトウェアです。

Uuidを作成するライブラリ Rpcrt4.lib の`UuidToString()`においてMBCS(マルチバイト文字セット)に対応した。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express

# ライセンス #

このソフトウェアはCC0ライセンスです。
